# RAG and Document Intelligence

Retrieval-augmented generation is still one of the most useful patterns for AI systems that need to answer from external knowledge.

## The core idea
- Store or index source material.
- Retrieve the most relevant parts for a question.
- Generate an answer grounded in that retrieved context.

## What has changed recently
- The ecosystem is moving from only vector search toward more agentic retrieval and document reasoning.
- Long-document tools are increasingly emphasizing traceability, citations, and auditability.
- PageIndex is a good example of this shift because it focuses on reasoning over document structure instead of only chunk similarity.
